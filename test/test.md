
**简要描述：** 

- 录制视频播放过程中，画人脸框

**请求URL：** 
- `http://{ip:port}/draw/faceframe`
  
**请求方式：**
- GET 

**参数：** 
- 表单

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|userid |是  |string | |
|sessionid |是  |string | |
|ph_path|是  |string |  |
|timestamp|是  |string | 前端发送的命中时间点 |

- 后台返回 timestamp 命中的一段数据

**请求示例**
```
http://10.12.23.6:8889/draw/faceframe?userid=soffeeshu1@tvu.com&sessionid=BB886A06967B4B49BC4590036357E89E&ph_path=mp4%2F598392A3A6D2D5FA0000000000000001%2F20190912-10-23-50_598392A3A6D2D5FA0000000000000001_9767A4DC0CC2F7E6_9577cf08-984c-442d-87c6-7f097fdd15bb%2F1%2Fstream.mpd&timestamp=1568283858513
```

|参数名|类型|说明|
|:-----  |:-----|-----                           |
| code |int   |errcode|
| message |string  |request message|
